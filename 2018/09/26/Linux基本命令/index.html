cd跳转命令 Change Directory

翻译过来就是改变文件夹 ,cd后面要跟一个文件夹的名字

  • 输入文件夹名字时支持tab补齐
  • 两个点表示跳到上一级文件夹中 eg: cd ..

cd跳转又分为相对路径和绝对路径

  • 相对路径

    打开当前文件夹里面的文件夹,或者 cd .. 跳到上一个文件夹

  • 绝对路径

    使用绝对路径最大的好处是跟当前位置无关 以老祖宗文件夹 / 打头,例如：不管我们在哪 cd / 都可以跳到根目录中, cd /Users/wenxiangye 可以直接跳到用户主目录 等价于 cd ~.
